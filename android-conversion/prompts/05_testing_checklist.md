# AI Studio 模擬器測試清單

在 AI Studio 右側的瀏覽器模擬器生成 App 後，請逐一檢查以下項目：

- [ ] 1. 畫面佈局：確認沒有 UI 元件重疊或超出螢幕邊界。
- [ ] 2. 資料載入：預設城市（例如台北）的天氣與 7 天預報是否有成功載入，並非顯示 Loading 或 Error。
- [ ] 3. 搜尋功能：點擊搜尋框輸入「台中」，確認下拉選單正確出現。點擊後是否切換到台中的天氣。
- [ ] 4. 橫向捲動：用滑鼠在 24 小時預報區塊橫向拖曳，確認可順利捲動 (LazyRow)。
- [ ] 5. 深淺色切換：點擊右上角的 Theme Toggle 按鈕，確認顏色正確切換，且 Glassmorphism 毛玻璃效果依然清晰。
- [ ] 6. 動畫展示：若當前天氣為雨天，確認畫面上有畫出雨滴落下效果。
