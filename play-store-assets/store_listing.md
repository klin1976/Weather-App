Weather App - Realtime Forecast

**Short Description (max 80 chars):** 
Real-time weather, 24h forecast, UV index, AQI, and voice broadcast in dark mode.

**Full Description (max 4000 chars):**
Discover the ultimate Weather App designed to provide you with real-time, accurate weather forecasts and environmental data in a beautiful, user-friendly interface. 

Features:
- **Real-Time Weather:** Get instant updates on current temperature, weather conditions, and more.
- **24-Hour & 7-Day Forecast:** Plan ahead with precise hourly and weekly weather predictions.
- **Air Quality Index (AQI):** Stay informed about the air you breathe with detailed AQI metrics.
- **UV Index:** Protect your skin with real-time UV index tracking and alerts.
- **Voice Broadcast:** Listen to the latest weather updates hands-free using our Web Speech API integration.
- **Temperature Trend Charts:** Visualize weather changes seamlessly with interactive Chart.js graphs.
- **Screenshot Sharing:** Easily capture and share beautiful weather cards with friends via html2canvas.
- **Beautiful Dark Mode UI:** Enjoy a premium, eye-catching design with smooth animations that look great in any lighting.

Our app prioritizes your privacy—location data is only used to fetch local weather and is never stored or sold. Experience weather tracking like never before!

**完整說明 (最長 4000 字元):**
探索這款為您量身打造的極致天氣應用程式，在精美易用的介面中為您提供即時、準確的天氣預報與環境數據。

主要功能：
- **即時天氣資訊：** 立即獲取當前溫度、天氣狀況等詳細資訊。
- **24 小時與 7 天預報：** 提供精準的逐時與一週天氣預測，讓您提早規劃行程。
- **空氣品質 (AQI)：** 詳細的空氣品質指標，隨時掌握您呼吸的空氣狀況。
- **紫外線指數 (UV Index)：** 即時追蹤紫外線指數，保護您的肌膚免受傷害。
- **語音播報功能：** 整合 Web Speech API，提供免持的天氣語音播報服務。
- **溫度趨勢圖：** 透過互動式的 Chart.js 圖表，輕鬆視覺化天氣變化趨勢。
- **精美截圖分享：** 內建 html2canvas 截圖功能，一鍵捕捉並與好友分享精緻的天氣卡片。
- **絕美的深色模式 UI：** 享受具備流暢動畫的高質感設計，無論任何光線下都令人驚豔。
